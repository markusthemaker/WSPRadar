# WSPRadar Tools

Small helper utilities that support WSPRadar workflows but are not part of the Streamlit app runtime.

## Timed A/B Relay Switch

Cross-platform Python console helper for selecting USB HID relay paths from a shared Repeat Interval and disjoint Target Start / Reference Start phases. This is the recommended helper for Windows, Linux and macOS timed hardware A/B switching; manual physical relay ON/OFF control remains available after setup.

Location: `tools/Timed-AB-Relay-Switch`
